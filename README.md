# App Screen Shot

모바일 앱 화면 녹화 동영상을 업로드하면 브라우저에서 주요 화면 전환 지점을 찾아 PNG 스크린샷으로 추출하는 로컬 웹앱입니다.

## 실행

```bash
npm install
npm run dev -- --host 127.0.0.1
```

브라우저에서 `http://127.0.0.1:5173/`을 열고 MP4, MOV, WebM 화면 녹화 파일을 업로드하면 됩니다.

## 동작 방식

- 동영상을 서버에 올리지 않고 브라우저 안에서 처리합니다.
- 설정한 간격으로 프레임을 읽고, 이전 주요 화면과의 이미지 차이가 임계값 이상일 때 캡쳐합니다.
- 결과 이미지는 개별 PNG 또는 ZIP 파일로 다운로드할 수 있습니다.

## 참고

iPhone 화면 녹화처럼 HEVC로 저장된 MP4는 브라우저/OS의 재생 지원 여부에 영향을 받습니다. 영상 미리보기가 재생되지 않으면 H.264 MP4로 변환한 뒤 업로드하세요.
