# App Screen Shot

모바일 앱 화면 녹화 영상에서 주요 화면 전환 지점을 찾아 PNG 스크린샷으로 추출하는 로컬 웹앱입니다. 영상은 외부 서버로 업로드하지 않고 브라우저 안에서만 처리합니다.

## 주요 기능

- MP4, MOV, WebM 등 브라우저가 재생할 수 있는 동영상 업로드
- 프레임 차이를 분석해 중복 화면을 제외하고 주요 화면 자동 캡쳐
- 샘플 간격, 화면 변화 민감도, 최소 캡쳐 간격, 최대 캡쳐 수 조정
- 추출된 화면 미리보기, 개별 PNG 다운로드, 선택/전체 ZIP 다운로드
- Figma 플러그인에서 재생 중 프레임 체크포인트 생성, 메모, 타임코드 이동 및 원본 PNG 추출
- Figma 플러그인 창 크기 드래그 조절 및 기본 크기 복원
- Figma 개발 플러그인으로 같은 추출 기능을 Figma 안에서 사용 가능

## 실행 방법

```bash
npm install
npm run dev -- --host 127.0.0.1
```

브라우저에서 `http://127.0.0.1:5173/`을 열고 화면 녹화 파일을 업로드합니다.

## 빌드

```bash
npm run build
npm run preview
```

빌드 결과물은 `dist/`에 생성됩니다. 이 폴더는 Git 추적 대상에서 제외되어 있습니다.

## 사용 흐름

1. 화면 녹화 파일을 업로드합니다.
2. 필요하면 왼쪽 패널에서 추출 설정을 조정합니다.
3. `추출 시작`을 누릅니다.
4. 자동으로 감지된 화면을 확인하고 필요한 이미지만 선택합니다.
5. 개별 PNG 또는 ZIP 파일로 다운로드합니다.

## 추출 설정

| 설정 | 설명 |
| --- | --- |
| 샘플 간격 | 몇 초마다 프레임을 읽을지 정합니다. 값이 작을수록 더 촘촘하게 분석하지만 처리 시간이 늘어납니다. |
| 화면 변화 민감도 | 이전 화면과 얼마나 달라야 새 화면으로 판단할지 정합니다. 값이 낮을수록 더 많은 화면이 캡쳐됩니다. |
| 최소 캡쳐 간격 | 너무 가까운 시간대의 중복 캡쳐를 방지합니다. |
| 최대 캡쳐 수 | 한 번에 저장할 수 있는 최대 화면 수입니다. |

## Figma 플러그인

`figma/` 폴더에는 Figma Desktop에서 불러올 수 있는 개발용 플러그인이 포함되어 있습니다.

설치 방법:

1. Figma Desktop을 엽니다.
2. `Plugins > Development > Import plugin from manifest...`를 선택합니다.
3. `figma/manifest.json` 파일을 선택합니다.

플러그인에서는 영상에서 추출한 화면을 Figma 페이지에 원본 크기 이미지 보드로 배치할 수 있습니다. 자세한 내용은 `figma/README.md`를 참고하세요.

## 프로젝트 구조

```text
.
├── assets/          # README 및 문서용 이미지
├── figma/           # Figma 개발 플러그인
├── qa/              # 화면 확인용 스크린샷
├── src/             # React 앱 소스
├── index.html       # Vite 진입 HTML
├── package.json     # 실행 스크립트와 의존성
└── README.md
```

## 주의사항

- 영상 처리는 브라우저의 동영상 디코딩 지원 범위에 영향을 받습니다.
- iPhone 화면 녹화처럼 HEVC로 저장된 MP4가 재생되지 않으면 H.264 MP4로 변환한 뒤 업로드하세요.
- 긴 영상이나 고해상도 영상은 분석 시간이 길어질 수 있습니다.
- 추출 결과는 브라우저 메모리 안에 임시로 보관되므로 필요한 파일은 다운로드해 보관하세요.
